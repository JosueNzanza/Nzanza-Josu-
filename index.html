COMPÉTENCE

Conception de l'interface utilisateur

Sur le Web, la conception d'interface utilisateur ou l'ingénierie d'interface utilisateur est la conception d'interfaces utilisateur pour les machines et les logiciels, tels que les ordinateurs, les appareils électroménagers, les appareils mobiles et autres appareils électroniques, en mettant l'accent sur l'optimisation de la convivialité et de l'expérience utilisateur.

DÉVELOPPEMENT WEB

FRONT END

Le développement Web frontal est le développement de l'interface utilisateur graphique d'un site Web grâce à l'utilisation de HTML, CSS et JavaScript afin que les utilisateurs puissent visualiser et interagir avec ce site Web.
